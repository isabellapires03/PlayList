
# Projeto Disciplina: Front-End


Olá! Este repositório faz parte do projeto da disciplina de Front-End da UTFPR - Campus Cornélio Procópio. 

***1. Objetivo***

Este projeto busca colocar em prática medidas de integração do HTML e CSS, com a escolha de uaminteface do tipo vitrine da escola dos alunos.
O tipo de vitine escolhida foi: Um site para a visuzalização de albuns de música!
